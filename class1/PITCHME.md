### Math Operators
You should give a and b a value and try out Python as a calculator

```python
# take a to the power of b
a**b
# modulus (remainder)
a%b
# Integer division (floored quotient)
a//b
# Division
a/b
# Multiplication
a*b
# Substraction
a-b
# Addition
a+b
```
